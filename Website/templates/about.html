About Us

Our irrigation system is designed to support better water management for plants. It reads sensor data from the field and displays it in a simple dashboard so the user can understand plant conditions quickly.

Live Monitoring

Soil moisture, temperature, and humidity data is fetched from Blynk.

Secure Dashboard

Only logged-in users can open the dashboard and view live sensor values.

Water Saving

The system helps users make better irrigation decisions for plants.
